# Git Analogy

Git versions files inside repos.

Timpos versions context paths inside workstreams.

| Git | Timpos |
|---|---|
| Repo | Workstream |
| File path | Path |
| File content | Value |
| Commit-like record | Moment |
| Commit parent | Previous |
| Commit metadata/source | Timpo |
| Checkout | Replay |
| Diff | Diff |

## Core distinction

Git records code history.

Timpos records source-located context state history.

## Keeper

```text
Timpo says when and where.
Moment says what changed.
```
